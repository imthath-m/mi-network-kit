A small libray to remove the network layer in your Swift application. 

## Installation
### Cocoapod
Add the following line to your podfile
pod 'MINetwrokKit'

### Swift Package
It is now also available as a Swift Package at https://github.com/imthath-m/mi-network-kit

## Reference
